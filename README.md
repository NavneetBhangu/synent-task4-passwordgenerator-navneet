# 🔐 Enhanced Password Generator

A professional Python-based Password Generator developed as part of the **Python Development Internship Program at Synent Technologies**.

## 📌 Project Overview

This project generates strong and secure passwords based on user preferences. Users can customize password creation by choosing character types such as uppercase letters, lowercase letters, numbers, and symbols.

The program also evaluates password strength as **Weak, Medium, or Strong**.

## ✨ Features

✅ User-defined password length
✅ Include uppercase letters
✅ Include lowercase letters
✅ Include numbers
✅ Include special characters
✅ Password strength checker
✅ Input validation and error handling
✅ Interactive Command Line Interface (CLI)

## 🛠️ Technologies Used

* Python
* Random Module
* String Module
